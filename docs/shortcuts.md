# 隐藏快捷键与作弊码 (Cheatsheet)

为了方便开发者调试和快速测试宠物在不同成长阶段和数据下的表现，我们在主进程（`src/windows/main/main.js` 和 `src/ini/sys.js`）中内置了一批隐藏的快捷键（作弊码）。

## 属性修改类快捷键 (作弊码)
在企鹅保持焦点或处于后台运行时（依赖全局快捷键拦截器），可以通过以下组合键快速修改宠物核心属性：

| 快捷键 | 功能 | 说明 |
| --- | --- | --- |
| `Ctrl` + `Shift` + `3` | **加钱 (元宝 +100)** | 快速增加 100 个元宝，方便去商城消费 |
| `Ctrl` + `Shift` + `4` | **扣钱 (元宝 -100)** | 减少 100 个元宝 (最低为 0) |
| `Ctrl` + `Shift` + `2` | **长经验 (成长值 +5000)** | 快速增加 5000 点成长值，极速**跨越年龄阶段**（蛋 -> 幼年 -> 成年），触发升级特效 |
| `Ctrl` + `Shift` + `1` | **掉经验 (成长值 -5000)** | 扣除 5000 点成长值 (最低为 0) |

## 物品获取类快捷键 (作弊码)
使用小键盘区的符号按键，可以随机从物品库中获得指定的物资：

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl` + `Shift` + `Numpad /` | **随机获得一个“日用品”** (香皂、毛巾等) |
| `Ctrl` + `Shift` + `Numpad *` | **随机获得一个“食物”** (汉堡、果汁等) |
| `Ctrl` + `Shift` + `Numpad -` | **随机获得一个“药品”** (感冒药、消炎药等) |
| `Ctrl` + `Shift` + `Numpad +` | **一键清空背包** (清理所有的当前持有物品) |

## 系统及控制类快捷键
| 快捷键 | 功能 |
| --- | --- |
| `Ctrl` + `Up/Down/Left/Right` (方向键任意)| **触发上帝模式 (God Mode)** 气泡对话 |
| `Alt` + `Q` | 预留的截图 (Screenshot) 功能触发按键 |
| `Alt` + `D` | 快速打开设置 (Settings) 窗口 |
| `Alt` + `Esc` | 关闭悬浮挂件工具窗口 (floatStyle) |
| `Alt` + `Shift` + `Ctrl` + `R` | **重载所有窗口 (Reload)**，快速刷新 UI 而无需完全重启应用 |

*注意：随意使用作弊码可能导致宠物状态异常（如瞬间衰老或直接饿死），如果出现 UI 动画不同步的情况，可以使用重载快捷键刷新。*
